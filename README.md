puppet-cloud-serverspec
=======================

The serverspec tests for the puppet-cloud modules

Getting Started
--------------

 * Install rubygems

  `yum install rubygems`

 * Install rake

   `gem install rake`

 * Installl serverspec

   `gem install serverspec`

 * Create a `arch.yml` file

 * Run `rake spec`
